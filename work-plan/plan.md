# 모바일 청첩장 제작 프로젝트 계획

## 킥오프 회의 기반 트렌드 요약
- **미니멀리즘 & 갤러리형:** 화이트 바탕에 깔끔한 폰트, 사진 중심의 레이아웃
- **모션 & 인터랙션:** 스크롤 시 부드럽게 나타나는 Fade-in 효과, 백그라운드 음악/영상 재생
- **필수 편의 기능:** 참석 여부(RSVP), 계좌번호 복사, 지도 API(카카오/네이버), 카카오톡 공유 썸네일 최적화

## 전체 제작 계획 (진행 예정)
- [ ] **1단계 [기본 구조]:** 반응형 웹 기본 마크업 및 미니멀리즘 UI 디자인 적용 (메인, 모시는 글, 갤러리)
- [ ] **2단계 [인터랙션]:** 동적 스크롤 모션 적용, 영상/배경음악 컴포넌트 추가
- [ ] **3단계 [핵심 기능]:** 지도 API 연동, 길안내, 송금 팝업 및 계좌 복사 버튼
- [ ] **4단계 [부가 기능]:** 방명록, 네이버/카카오톡 공유 최적화 (Open Graph 태그 등)

## 추가/변경 기획 내용 (2024 트렌디 밤하늘 컨셉)
- **디자인 컨셉:** 검정 배경에 귀여운 별모양 도트가 띄엄띄엄 배치된 트렌디한 감성
- **타이포그래피:** 별빛 배경과 어울리는 세련되고 트렌디한 폰트 적용
- **갤러리 UI:** 사진을 넘겨서 볼 수 있는 슬라이드(캐러셀) 방식 채택
- **인터랙션/이펙트:** 모달 버튼(예: 청첩장 하단 축하 버튼 등) 클릭 시 화면에 하트가 날아다니는 동적 효과 추가

## 추가 기획 내용 (웹 호스팅 및 배포)
- **5단계 [웹 배포]:** Vercel, GitHub Pages 등의 호스팅 서비스를 이용해 모바일/PC 상관없이 외부 접속이 가능한 고유 웹 링크(URL) 발급
- [선택 1] **Vercel 배포**: 가이드에 따라 가입 후 터미널 명령어만으로 매우 빠르고 쉽게 배포 가능합니다. (`https://원하는이름.vercel.app`)
- [선택 2] **GitHub Pages 배포**: GitHub 계정에 코드 보관 및 웹 호스팅을 동시에 진행합니다. (`https://아이디.github.io/프로젝트명`)

## 추가 기획 내용 (디자인 개편 및 방명록 추가)
- **6단계 [디자인 전면 개편]:** 다크 별빛 테마 → 크림/화이트 엘레강트 테마로 전환 (style.css 전면 재작성)
- **갤러리 변경:** Swiper 카드스택 → 2열 그리드 썸네일 + 이전/다음 라이트박스 모달
- **방명록 추가:** Firebase Firestore 연동, 이름+메시지 입력, 실시간 목록 표시
- **Firebase 설정 대기 중:** script.js 내 firebaseConfig에 실제 값 입력 필요
